# personal-finance-agent

• Designed and deployed an AI-powered personal finance assistant using Streamlit and Groq-powered LLaMA 3.3.
• Implemented features including expense analysis, financial health scoring, budgeting guidance, savings recommendations, and expense forecasting through interactive dashboards.
• Integrated CSV-based transaction analysis and document-assisted financial guidance to provide practical, user-focused insights.
• Deployed the application on Streamlit Cloud with GitHub-based version control, demonstrating end-to-end AI product development.
• Live Demo: https://finance-guide-ai.streamlit.app/
